# MERN Task Manager

This is a simple Task Manager application built using the MERN stack.

## Features
- User registration and login
- JWT-based authentication
- Create, view, update, and delete tasks
- Each user can access only their own tasks
- Change password functionality

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- Authentication: JWT

## Node Version
This project was built and tested using: Nodejs v20.12.2

## How to Run the Project

### Backend Setup (Root Folder)
1. Install dependencies:
   npm install
2. Create a `.env` file with:
   - MONGO_URI
   - JWT_SECRET
3. Start the server:
   npm start

### Frontend
1. Navigate to the frontend folder
2. Install dependencies:
   npm install
3. Start the React app:
   npm start

## Notes
- APIs were tested using Postman
- This project focuses on core functionality without additional libraries
